### **5.1. Основная теорема алгебры**
* **Формулировка:** всякий многочлен степени n с комплексными коэффициентами имеет ровно n корней с учетом кратности
* **Разложение над ℂ:** P(z) = aₙ∏(z - zᵢ)
* **Следствия для многочленов с вещественными коэффициентами:**
  * Комплексные корни идут парами (a±bi)
  * Разложение над ℝ: линейные и неприводимые квадратные множители

### **5.2. Теорема Виета**
* **Формулы Виета для корней:**
  * ∑xᵢ = -a_{n-1}/aₙ
  * ∑xᵢxⱼ = a_{n-2}/aₙ
  * ∏xᵢ = (-1)ⁿa₀/aₙ
* **Элементарные симметрические многочлены**
* **Выражение степенных сумм через элементарные симметрические многочлены**
* **Формулы Ньютона для степенных сумм**

## **VII. БЫСТРОЕ УМНОЖЕНИЕ МНОГОЧЛЕНОВ**

### **7.1. Классические алгоритмы**
* **Наивный алгоритм:** O(n²)
* **Алгоритм Карацубы:** O(n^{log₂3}) ≈ O(n^{1.585})
  * Рекурсивное разбиение
  * Оптимизация через 3 умножения вместо 4
* **Алгоритм Тома-Кука (Toom-Cook):** O(n^{1+ε})
  * Обобщение Карацубы
  * Toom-3, Toom-4

### **7.2. Преобразование Фурье**
* **Дискретное преобразование Фурье (DFT):**
  * Определение: A(ωₖ) = ∑ aⱼ ωₖʲ, где ω = e^{2πi/n}
  * Свойства: линейность, свертка → произведение
* **Быстрое преобразование Фурье (FFT):**
  * Алгоритм Кули-Тьюки (Cooley-Tukey)
  * Рекурсивная и итеративная реализации
  * Bit-reversal permutation
* **Обратное преобразование Фурье (IFFT)**

### **7.3. Теоретико-числовое преобразование (NTT)**
* **Преобразование в конечных полях**
* **Требования к модулю:** существование примитивного корня степени 2^k
* **Модули для NTT:** 998244353, 1004535809, 469762049
* **Китайская теорема об остатках для больших коэффициентов**

### **7.4. Продвинутые алгоритмы**
* **Алгоритм Шёнхаге-Штрассена:** O(n log n log log n)
* **Алгоритм Фюрера:** O(n log n 2^{O(log* n)})
* **Алгоритмы для умножения разреженных многочленов**
