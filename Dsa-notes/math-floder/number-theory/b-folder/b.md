## **B. Модулярная арифметика и Диофантовы уравнения**

### **Действия + - / * по модулю**
* (a+b)%m = (a%m + b%m)%m
* (a-b+m)%m = (a%m - b%m + m)%m
* (a*b)%m = (a%m * b%m)%m
* (a/b)%m = (a%m * b^(-1)%m)%m

### **7.1. Обратные элементы**
* a⁻¹ (mod m) существует ⇔ gcd(a,m) = 1
* **Вычисление:** через расширенный алгоритм Евклида

### **7.2. Китайская теорема об остатках**
* **Система:** x ≡ aᵢ (mod mᵢ), где gcd(mᵢ, mⱼ) = 1 для i ≠ j
* **Решение:** x = ∑ aᵢ Mᵢ Nᵢ (mod M), где M = ∏ mᵢ, Mᵢ = M/mᵢ, Nᵢ ≡ Mᵢ⁻¹ (mod mᵢ)

## **7.3. Теоремы**
* **Малая теорема Ферма:** a^{p-1} ≡ 1 (mod p) для простого p и p ∤ a
* **Теорема Эйлера:** a^{φ(m)} ≡ 1 (mod m) при gcd(a,m) = 1
* **Функция Эйлера:** φ(n) = n ∏_{p|n} (1 - 1/p)

# **IX. ТЕОРИЯ СРАВНЕНИЙ**

## **9.1. Линейные сравнения**
* ax ≡ b (mod m) имеет решение ⇔ gcd(a,m) | b
* **Количество решений:** gcd(a,m) при условии существования
* **Алгоритм:** приведение к диофантову уравнению

## **9.2. Системы сравнений**
* **Теорема:** система x ≡ aᵢ (mod mᵢ) совместна ⇔ aᵢ ≡ aⱼ (mod gcd(mᵢ, mⱼ)) ∀i,j
* **Алгоритм:** последовательное слияние сравнений

## **9.3. Степенные сравнения**
* xᵏ ≡ a (mod m)
* **Проверка существования:** через индексы и первообразные корни
* **Алгоритм:** подъем решений по степеням простых

# **X. ДИОФАНТОВЫ УРАВНЕНИЯ**

## **10.1. Линерные уравнения**
* ax + by = c имеет целые решения ⇔ gcd(a,b) | c
* **Общее решение:** если (x₀, y₀) — частное, то:
    - x = x₀ + (b/d)t
    - y = y₀ - (a/d)t, где d = gcd(a,b), t ∈ ℤ

## **10.2. Уравнение Пелля**
* x² - Dy² = 1, где D — не квадрат
* **Фундаментальное решение:** минимальное положительное
* **Общее решение:** через степени фундаментального

## **10.3. Пифагоровы тройки**
* x² + y² = z²
* **Примитивные тройки:** x = m² - n², y = 2mn, z = m² + n², где gcd(m,n)=1, m>n>0, разной четности

# **XII. ТЕОРЕТИКО-ЧИСЛОВЫЕ ФУНКЦИИ**

## **12.1. Функция Кармайкла λ(n)**
* **Определение:** наименьшее m такое что aᵐ ≡ 1 (mod n) для всех a с gcd(a,n)=1
* **Свойства:** λ(pᵏ) = φ(pᵏ) для нечетных простых, λ(2) = 1, λ(4) = 2, λ(2ᵏ) = 2^{k-2} при k≥3

## **12.2. Радикал числа**
* rad(n) = ∏_{p|n} p (произведение различных простых делителей)
* **ABC-гипотеза:** связь a+b=c с rad(abc)

## **12.3. Функция Мангольдта**
* Λ(n) = ln p если n = pᵏ, иначе 0
* **Связь:** ∑_{d|n} Λ(d) = ln n

# **XIV. ВАЖНЫЕ ТЕОРЕМЫ**

1. **Евклида:** простых чисел бесконечно много
2. **Дирихле:** в арифметической прогрессии a + nd при gcd(a,d)=1 бесконечно много простых
3. **Вильсона:** (p-1)! ≡ -1 (mod p) для простого p
4. **Ферма о суммах квадратов:** p = 2 или p ≡ 1 (mod 4) ⇔ p = a² + b²
5. **Лагранжа:** полиномиальное сравнение степени n имеет не более n решений по простому модулю
6. **Чебышёва:** 7/8 < π(x)/(x/ln x) < 9/8 для больших x
